PokeDex App

This PokeDex was created using an existing online API. The app was formatted using Bootstrap and a mobile first design concept. About page and GitHub link available within the app.

Meet App

This Progressive Web Application allows the user to search for web development classes being offered by CareerFoundry close to their city.

  • Tech Used:
  • React.js
  • axios and async/await
  • ReCharts Library for Data Visualization
  • Serverless Authorization (AWS lambda and Google)
  • Service Workers for offline access
  • Test Driven Development: Testing with JEST and Cucumber
  • Unit, Integration and End to End testing
  • Gherkin Syntax
  • Object Oriented Programming for User Alert Messages
  • Application performance monitoring by Atatus
